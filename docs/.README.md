## ğŸš€ Building the Docs

### Prerequisites
- Install GitBook CLI or use GitBook.com

### Local Development
```bash
# Install GitBook (if not already installed)
npm install -g gitbook-cli

# Navigate to docs directory
cd docs

# Install plugins
gitbook install

# Serve locally
gitbook serve

# Open browser to http://localhost:4000
```

### Build Static Site
```bash
cd docs
gitbook build

# Output in _book/ directory
```

## ğŸ“– GitBook.com Deployment

### Option 1: GitBook Cloud
1. Sign up at [gitbook.com](https://www.gitbook.com)
2. Connect GitHub repository
3. Select `docs/` as documentation root
4. GitBook auto-syncs on push

### Option 2: Manual Upload
1. Build locally: `gitbook build`
2. Upload `_book/` directory to hosting
